This website is hosted via Replit, and uses Node.js with Express.js and EJS. Express is an web hoster on Node, and EJS is something to make templates.
FluffyScratch is used for the login system.
The Scratch api, Scratch Uploads, and the abilaty to embed a Scratch project were used, too.
ReplitDB will be used at somepoint in the future to get data from the backend, as well as send data to it.
